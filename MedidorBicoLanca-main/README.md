# MedidorBicoLanca

Repositório referente aos códigos do sistema medidor de desgaste em furo de bico de lança de convertedores LD. O sistema proposto, baseado em Yolact, visa segmentar e realizar a medição dos furos, permitindo melhor e mais rápido controle da manutenção da lança.
